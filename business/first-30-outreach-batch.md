# First 30 Outreach Batch

Use this when the family wants the fastest path from "we have a website" to "people are seeing the booking page."

Do not ask Dad or Mom to run this. One person can send these messages from Instagram, website contact forms, or public contact pages. Every reply goes into `lead-tracker.csv`.

Main booking link:

`https://miamipapanoel.com/book?source=first-30-outreach`

Proof link:

`https://miamipapanoel.com/reviews?source=first-30-outreach`

## Tracking Rule

For every message sent, add one row to `lead-tracker.csv`:

- Lead name
- Lead type
- Contact method
- Date sent
- Status: New lead
- Next action: Follow up in 3 days

## Follow-Up Message

If there is no reply after 3 days:

```text
Hi, just following up once in case a bilingual Santa visit could be helpful for your families, residents, clients, or guests. No pressure at all. The booking request page is here if you ever need it:
https://miamipapanoel.com/book?source=outreach-followup
```

## Stop Rule

Stop after one follow-up unless they reply. The goal is warm, respectful visibility, not spam.
